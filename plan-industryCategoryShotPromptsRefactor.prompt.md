## Plan: Industry-Category-Shot Prompt Refactor (DRAFT)

This change introduces a hierarchical prompt model where each generated image uses: reference images + industry prompt + category prompt + shot prompt, while keeping max outputs at 4 and preserving provider-specific behavior (OpenAI Batch multi-shot in one batch job, REVE one shot per API call). Based on your decisions, batch runs use one industry for the full batch, categories come from the Excel category column (or dropdown for folder upload), category catalogs are admin-managed fixed lists, overrides are supported at industry+category+shots granularity, and selected shot count must exactly equal requested image count.
